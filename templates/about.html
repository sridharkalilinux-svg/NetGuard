How Detection Works

  • Reconnaissance Detects port scanning (High SYN, Nmap patterns).
  • Brute Force Identifies repeated authentication failures on SSH/FTP/HTTP.
  • Injections Scans payloads for SQLi (`UNION SELECT`) and Command Injection patterns.
  • Clear-Text Creds Extracts unencrypted Basic Auth and FTP credentials.

Core Concepts

PCAP Tracing

Packet Capture (PCAP) is an API for capturing network traffic. It logs the exact data packets traversing a network interface, allowing analysts to reconstruct sessions, inspect payloads, and diagnose connectivity issues or security breaches.

Cybersecurity & Forensics

Network forensics involves monitoring and analyzing computer network traffic for the purposes of information gathering, legal evidence, or intrusion detection. Unlike firewalls which block traffic, forensics tools analyze what has already passed through.

Network Protocols

Standardized rules for formatting data (like TCP, UDP, HTTP). Analyzing protocol headers helps identify anomalies, such as non-standard flags in TCP handshakes or malformed HTTP requests indicative of attacks.

Industry Context

BlueHat

Refers to external security researchers or Microsoft's security conference series. In a broader context, it represents the collaboration between independent researchers and software vendors to identify and patch vulnerabilities before they are exploited.

Understanding Reconnaissance

Reconnaissance is the preliminary phase of a cyber attack where the adversary gathers information about the target system or network. This stage is critical as it defines the strategy for the subsequent intrusion. Unlike brute-force or exploitation attacks, reconnaissance can often be "silent," making it harder to detect without deep packet inspection.

Key Objectives
  • Asset Discovery

    Identifying active hosts, IP addresses, and network topology to map the attack surface.

  • Service Enumeration

    Determining which applications and versions are running on open ports (e.g., Apache 2.4, SSH-2.0).

  • Vulnerability Mapping

    Correlating discovered services with known CVEs to find entry points.

NetGuard Detection Strategy

NetGuard utilizes heuristic analysis to spot reconnaissance patterns that evade simple firewall rules:

Port Scanning (Nmap) High Noise

Detects rapid connection attempts across multiple ports from a single source.

Stealth Scans (SYN/FIN) Low Profile

Identifies incomplete TCP handshakes used to map ports without establishing full connections.
